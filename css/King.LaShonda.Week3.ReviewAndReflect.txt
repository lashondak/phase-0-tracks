What is the box model?
The basic structure of a CSS style sheet. It is the basically the notion that a box wraps around every HTML element.


Explain and give an example of the cascading effect of CSS.
Cascading means that you can cover (apply) multiple style sheets to one HTML document. If you have two different .css files, you can apply them to one HTML page.
